# Installation

TIP

machinable requires Python 3.7 or higher and does not support Python 2.

Install the latest stable version via pipopen in new window:

pip install machinable

# For development

To test or develop new features you may want to install the latest package version from the repository.

Clone the source from the public code repositoryopen in new window on GitHub and change into the machinable directory.

Install the development dependencies using poetryopen in new window:

poetry install -D

To build the Vuepressopen in new window-based documentation run:

npm install
npm run docs:dev

If you plan to contribute please read the contribution guide
