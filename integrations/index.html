Integrations ​

Community-maintained interfaces you can pull into a project by URL: execution backends, storage adapters, and reusable components. Each page has a copy-paste install snippet, and adding your own is a single pull request.

Community-maintained

Integrations are not part of machinable core: they are not covered by the core test suite or releases and come with no maintenance guarantee. Issues go to each integration's maintainer. See the policy and contribution process. Pulled modules execute arbitrary code on import, so review what you pull (machinable fetch <name> downloads without importing).
